# Glossary

## Key Terms

- **Agent**: An autonomous entity in PRYMA, capable of reasoning, acting, and evolving.
- **Container**: Modular unit hosting agents, protocols, and resources.
- **Lumen**: The central reasoning agent (see 03_agents/agent_types.md).
- **Strike**: The executor agent (see 03_agents/agent_types.md).
- **Echo**: The communications agent (see 03_agents/agent_types.md).
- **Sentinel**: The security agent (see 03_agents/agent_types.md).
- **Sage**: The planning agent (see 03_agents/agent_types.md).
- **Archive**: The knowledge agent (see 03_agents/agent_types.md).
- **Spirit**: The micro-agent, source of emergent behavior (see 03_agents/agent_types.md).
- **Shade**: The adversarial sandbox agent (see 03_agents/agent_types.md).
- **Protocol**: The set of rules governing agent and container interactions (see 04_protocol/pryma_protocol.md).
- **WASM**: Universal runtime for cross-language execution (see 05_runtime/wasm_spec.md).
- **Solana**: Blockchain platform for governance and identity (see 04_protocol/solana_integration.md).
